## ✨ Features

| Feature | Description |
|---|---|
| 🏦 **Healthcare Navigator** | AI decision intelligence with medical loan application (0% EMI), cost estimation & hospital discovery |
| 🤖 **AI Doctor** | Real-time streaming medical chat powered by Groq LLaMA 3 70B |
| 🩺 **Symptom Checker** | AI-powered differential diagnosis with severity scoring |
| 💊 **Medicine Scanner** | Camera-based medicine identification & analysis |
| 💊 **Medicine Finder** | Search 246K+ medicines with nearby pharmacy locator |
| 🧬 **Health Predictors** | 9 disease risk models (diabetes, cancer, dengue, kidney, etc.) |
| 🧪 **Diagnostic Centre** | Book 50+ lab tests & health packages with Razorpay payments |
| 🏥 **Hospital Finder** | Interactive map with 249K+ healthcare facilities (GPS-enabled) |
| 🧠 **AI Health Quiz** | AI-generated quizzes on any health topic |
| 👨‍⚕️ **Doctor Directory** | Specialist discovery & appointment booking |
| 📱 **8-Language Support** | English, Hindi, Telugu, Tamil, Kannada, Marathi, Bengali, Bhojpuri |

## 🚀 Quick Start

```bash
# Backend
cd backend && npm install
cp .env.example .env          # Configure environment variables
npm run seed                   # Seed 496K+ records
npm start                      # → http://localhost:5000

# Frontend
cd frontend && npm install
cp .env.local.example .env.local  # Configure environment variables
npm run dev                        # → http://localhost:3000
```

## 🛠 Tech Stack

**Frontend:** Next.js 16 • React 19 • TypeScript • Tailwind CSS 4 • Framer Motion • Leaflet Maps • Clerk Auth • Razorpay  
**Backend:** Express 5 • Supabase (PostgreSQL) • Redis • Groq AI (LLaMA 3) • Cloudinary • Socket.io  
**Deploy:** Vercel (frontend) + Render (backend)
